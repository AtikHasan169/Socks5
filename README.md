# Railway SOCKS Proxy – Multi User + Multi Upstream

This project hosts a **public SOCKS5 proxy** on Railway using `gost`.

## Features

✔ Unlimited usernames & passwords  
✔ Unlimited upstream SOCKS proxies  
✔ Load balancing / failover  
✔ Public endpoint accessible worldwide  
✔ All configuration done via Railway Environment Variables  
✔ No code editing required
